# The Nunchi Health Migration Project 
This is a publiclly deployed and available version of the Nunchi Health website, everything you see here has been written from scratch with code—which means no squarespace! This project may take some time so I created this page for anyone to see how far along progress has been with the project; as well as to see how different the final website will be in comparison to the Squarespace version. Squarespace has thousands  of lines of code so the final result will not be 1:1, but it'll be forever Nunchi.

The priority is accessibility, performance, responsiveness and becoming ad-grant compliant so that Nunchi Health benefits long-term with organic traffic rather then suffering potential losses every year while forking hundreds of dollars over to a third-party.. 

## Tech Stack
Astro.js, React.js + HTML & SCSS
Cloudflare Pages & Workers
Stripe Checkout for e-commerce functionality

Might shift to Payload CMS or Sanity Studio to help non-techincal staff design new pages or at the very least update existing ones

### How to get started?
npm run dev — in your terminal to run the webpage on a localhost server

Try your hand at replicating any of the pages currently up for grabs!
